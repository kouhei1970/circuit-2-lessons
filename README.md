# 電気回路Ⅱ — 触って学べるインタラクティブ授業資料

教科書の内容を「読むだけ」でなく、**自分で値を動かして確かめられる**教材へ。
複素平面のドラッグ、波形スライダー、採点つき練習を、ブラウザだけで。各ページに音声解説つき。

🔗 **公開ページ:** https://kouhei1970.github.io/circuit-2-lessons/

---

## 収録教材

| 教材 | 回 | 内容 |

| [正弦波の複素数表示](lessons/lesson10/) | 第10回（8.7） | 正弦波を「回転する複素ベクトル（極形式）」で表す回。回転ベクトルの実部がコサイン波を描く同期アニメ、位相の進み・遅れ、sin→cos 変換、ベクトル加算による合成、例題8.8／8.9 を操作で確認。**全20ページ＋採点つき練習（課題1〜4）**。 |

## 使い方

- 各教材を開き、**→／Space** でページ送り（**←** で戻る）。
- 右上の **🔊 解説** でそのページの音声解説を再生（edge-tts による合成音声）。
- 右下の **⛶** でプロジェクタ向けの全画面表示。
- 練習問題は数値を入力して **採点**、**答えを見る**、**やり直す** が使えます。
- スライド直リンクに対応（URL 末尾 `#番号`）。

## 構成

```
index.html                 教材一覧（このページ）
lessons/lesson08/index.html  第8回 デッキ（+ audio/ 音声, narration.json, tts_dictionary.py）
lessons/lesson10/index.html  第10回 デッキ（+ audio/ 音声, narration.json, tts_dictionary.py）
```

各デッキは外部依存ゼロの単一HTML＋音声mp3で動きます（CDN・ビルド不要）。
